# IncomeBreakdownType

The type of income. Possible values include:
  `"regular"`: regular income
  `"overtime"`: overtime income
  `"bonus"`: bonus income


## Values

| Name                          | Value                         |
| ----------------------------- | ----------------------------- |
| `BONUS`                       | bonus                         |
| `OVERTIME`                    | overtime                      |
| `REGULAR`                     | regular                       |
| `LESS_THAN_NIL_GREATER_THAN_` | <nil>                         |
