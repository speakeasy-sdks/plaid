# PaymentChannel

The channel used to make a payment.
`online:` transactions that took place online.

`in store:` transactions that were made at a physical location.

`other:` transactions that relate to banks, e.g. fees or deposits.


## Values

| Name       | Value      |
| ---------- | ---------- |
| `ONLINE`   | online     |
| `IN_STORE` | in store   |
| `OTHER`    | other      |
