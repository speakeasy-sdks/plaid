# BankTransferEventListDirection

Indicates the direction of the transfer: `outbound`: for API-initiated transfers
`inbound`: for payments received by the FBO account.


## Values

| Name                          | Value                         |
| ----------------------------- | ----------------------------- |
| `INBOUND`                     | inbound                       |
| `OUTBOUND`                    | outbound                      |
| `LESS_THAN_NIL_GREATER_THAN_` | <nil>                         |
