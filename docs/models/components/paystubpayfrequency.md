# PaystubPayFrequency

The frequency at which the employee is paid. Possible values: `MONTHLY`, `BI-WEEKLY`, `WEEKLY`, `SEMI-MONTHLY`.


## Values

| Name           | Value          |
| -------------- | -------------- |
| `MONTHLY`      | MONTHLY        |
| `BI_WEEKLY`    | BI-WEEKLY      |
| `WEEKLY`       | WEEKLY         |
| `SEMI_MONTHLY` | SEMI-MONTHLY   |
