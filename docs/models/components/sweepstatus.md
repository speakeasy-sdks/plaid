# SweepStatus

The status of a sweep transfer

`"pending"` - The sweep is currently pending
`"posted"` - The sweep has been posted
`"settled"` - The sweep has settled
`"returned"` - The sweep has been returned
`"failed"` - The sweep has failed


## Values

| Name       | Value      |
| ---------- | ---------- |
| `PENDING`  | pending    |
| `POSTED`   | posted     |
| `SETTLED`  | settled    |
| `RETURNED` | returned   |
| `FAILED`   | failed     |
